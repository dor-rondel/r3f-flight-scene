# R3F Flight Scene

### Requirements

1. NodeJS v20+
2. NPM / Yarn / Deno / Bun etc

### Instructions

1. clone the repository
2. `cd` into the directory
3. install the dependencies with `npm i` or equivalent
4. run the project locally with `npm run dev`
